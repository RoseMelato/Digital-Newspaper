**Digital Newspaper**

A multi-page digital newspaper website built using HTML5 and CSS3.  
The project simulates a real-world online newspaper layout with a front page and multiple article pages.

**Project Overview**

The Digital Newspaper project was created to practice:
- Structuring multi-page websites
- Page navigation and layout design
- Styling content using CSS
- Organising media assets (images)

The project includes a front page and several article pages, each styled to resemble a modern digital newspaper.

**Technologies Used**

- HTML5  
- CSS3
  
**Features**

- Front page with highlighted news stories  
- Multiple article pages  
- Image-based news sections  
- Clean and readable layout  
- Linked pages for easy navigation
